
## Ambiente para execução 

 - Ruby 2.7 +
 - Bundler
 - Chromedriver

### Como instalar dependências do projeto

```
$ gem install bundler
```
```
$ bundler install
```
### Atualizar dependências do projeto ou sistema operacional
```
$ bundle update
```
```
$ gem update
```
